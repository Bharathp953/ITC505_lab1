Description of Finger Paint Font:

  • Carrois Apostrophe created the Finger Paint Font.
  • A paintbrush effect was used to create the beginning of this font and then it was transformed into a real typeface.
  • Websites for children and educational content most commonly use the Finger Paint font in HTML.
  • There is a license attached to this font called the Open Font License. In this way, you are able to use them in a wide range of products and projects, whether printed or digital, commercial or otherwise.

Process that I used to get Fonts in my WebPage:

      1. The following fonts were acquired from fonts.google.com
      2. Upon clicking on Finger paint font, I was taken to the Finger paint font page.
      3. As soon as I open a font, I see a button that says Get font in the top-right corner.
      4. In the right corner after you click the Get font button, there are two buttons titled Get embed code and Download all.
      5. In the menu that appeared, once I selected get embed code, there were a few options such as Web, Android, and I chose Web.
      6. That web has several radio buttons, including a link with HTML code and CSS classes.
      7. It was in my head tag as well as in my style sheet to include that HTML code.
